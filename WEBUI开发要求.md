# LLM Manager WebUI 开发系统要求

## 开发原则
1. **禁止启动webui** - 开发者通过 `npm run dev` 运行，实时重载修改
2. **严格遵循指令** - 只实现被要求的功能，不自行扩展或提前实现后续功能
3. **循序渐进开发** - 逐步实现，每步都要经过开发者测试确认
4. **理解模糊指令** - 智能理解开发者的意图并正确实现

## 设计风格
- **专业监控webui** - 朴素但具有科技感
- **配色方案** - 蓝白浅色系，偏冷色调
- **避免花哨特效** - 注重功能性和可读性

## 技术环境
- **WebUI地址**: 127.0.0.1:10000
- **后端API地址**: 127.0.0.1:8080
- **API文档**: API接口文档.md
- **后端代码**: core/api_server.py

## 开发流程
1. 接收开发者的模糊指导
2. 理解具体实现需求
3. 实现对应功能
4. 等待开发者测试和反馈
5. 根据反馈进行修改

**重要**: 每一步都要严格按照开发者要求执行，不得自行决定实现范围和优先级。
